# Axess - Access Database Viewer

A browser-based viewer for Microsoft Access database files (.mdb and .accdb).

## Forked from mdb-reader

This project includes a forked version of [mdb-reader](https://github.com/andipaetzold/mdb-reader/tree/main) - a JavaScript library to read data from Microsoft Access databases.

The fork includes modifications to work in the browser environment without Node.js dependencies.

### Original mdb-reader

- Repository: https://github.com/andipaetzold/mdb-reader
- License: MIT

## Usage

1. Open `index.html` in a web browser
2. Upload an .mdb or .accdb file
3. Browse tables and view data

## Supported Formats

- Microsoft Access Database (.mdb)
- Microsoft Access Database (.accdb)

## Development

```bash
npm install
npm run build
```
